# pict-section-theme

> Bundled Retold themes, drop-in theme controls, and build-time brand tooling for Pict applications

- Theme-picker dropdown grouped by category, with mode-capability glyphs
- Light / dark / system mode toggle that respects single-mode themes
- Viewport-scale select, independent of the active theme
- Embeddable SVG topbar button that pops a theme submenu via pict-section-modal
- Per-host persistence of theme, mode, and scale
- Build-time brand + favicon precompute CLIs
